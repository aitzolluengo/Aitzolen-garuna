# Advanced Reported Speech (B2)

El Reported Speech (o Indirect Speech) se usa para comunicar lo que otra persona dijo, pensó o preguntó, sin citar sus palabras exactas. En B2, vamos más allá de los cambios básicos de tiempo y pronombres.

## Repaso B1: Cambios Básicos

Cuando pasamos de Direct Speech a Reported Speech, generalmente:

*   **Tiempos Verbales:** Retroceden un tiempo (Present Simple -> Past Simple, Present Perfect -> Past Perfect, etc.).
*   **Pronombres y Posesivos:** Cambian según el contexto (I -> he/she, my -> his/her).
*   **Adverbios de Tiempo y Lugar:** Cambian (now -> then, here -> there, yesterday -> the day before, tomorrow -> the next day).
*   **Verbo Introductor:** Se usan *say* o *tell* (e.g., *He said that...*, *She told me that...*).

## Usos Avanzados en B2

### 1. Reporting Questions (Preguntas Reportadas)

*   **No se usa "do/did":** Se elimina el auxiliar `do/did`.
*   **Orden de Sujeto-Verbo:** El orden de la pregunta se convierte en el de una afirmación (Sujeto + Verbo).
*   **Con "if" o "whether":** Si la pregunta original es de Sí/No.
*   **Con "wh-words":** Si la pregunta original usa *what, where, when, why, who, how*.
*   **Verbos Introductorios:** *ask, wonder, enquire, want to know*.

    *   **Direct:** *"Are you coming?"*
    *   **Reported:** *He asked **if I was coming**.*
    *   **Direct:** *"Where do you live?"*
    *   **Reported:** *She wanted to know **where I lived**.*

### 2. Reporting Commands and Requests (Órdenes y Peticiones Reportadas)

*   **Verbos Introductorios:** *tell, order, ask, advise, warn, instruct, persuade, beg*.
*   **Estructura:** Verbo Introductor + Objeto + (not) + to-infinitivo

    *   **Direct:** *"Close the door!"*
    *   **Reported:** *He told me **to close the door**.*
    *   **Direct:** *"Don't touch that!"*
    *   **Reported:** *She warned him **not to touch that**.*
    *   **Direct:** *"Please help me."*
    *   **Reported:** *He asked me **to help him**.*

### 3. Reporting Suggestions (Sugerencias Reportadas)

*   **Verbos Introductorios:** *suggest, recommend, advise*.
*   **Estructura:** Verbo Introductor + that + Sujeto + (should) + base verb (o + Verbo -ing)

    *   **Direct:** *"Why don't we go to the cinema?"*
    *   **Reported:** *She suggested **that we (should) go to the cinema**.* (O: *She suggested **going to the cinema**.*)

### 4. Other Reporting Verbs (Otros Verbos de Reporting)

Hay muchos otros verbos que nos permiten reportar con más precisión la intención del hablante.

| Verbo         | Estructura               | Ejemplo (Reported)                                         |
| :------------ | :----------------------- | :--------------------------------------------------------- |
| **Agree**     | to + infinitive / that   | *He agreed to help.* / *He agreed that he would help.*      |
| **Admit**     | -ing / that              | *He admitted taking the money.* / *He admitted that he had taken the money.* |
| **Apologise** | for + -ing               | *She apologised for being late.*                           |
| **Complain**  | to + somebody + about + -ing / that | *He complained to the manager about the service.* / *He complained that the service was bad.* |
| **Deny**      | -ing / that              | *He denied stealing the car.* / *He denied that he had stolen the car.* |
| **Insist**    | on + -ing / that         | *She insisted on paying.* / *She insisted that she would pay.* |
| **Offer**     | to + infinitive          | *He offered to carry my bag.*                              |
| **Promise**   | to + infinitive / that   | *She promised to call.* / *She promised that she would call.* |
| **Refuse**    | to + infinitive          | *He refused to answer my question.*                        |
| **Remind**    | somebody + to + infinitive | *She reminded me to buy milk.*                             |
| **Suggest**   | -ing / that              | *He suggested visiting the museum.* / *He suggested that we visit the museum.* |
| **Threaten**  | to + infinitive          | *He threatened to call the police.*                        |
| **Warn**      | somebody + (not) + to + infinitive | *She warned me not to go there.*                           |
