# Report Structure (B2)

Un report (informe) es un documento formal que presenta información fáctica de manera clara y organizada, a menudo analizando una situación o problema y ofreciendo conclusiones y recomendaciones.

## Estructura General

### 1. Title (Título)
*   Debe ser claro y conciso, indicando el tema del informe (e.g., *Report on the feasibility of...*, *Investigation into the causes of...*).

### 2. Introduction (Introducción)
*   **Purpose:** Explica por qué se ha escrito el informe y qué pretende conseguir.
    *   *The aim of this report is to...*
    *   *This report intends to examine...*
*   **Scope:** Describe los aspectos que cubrirá el informe.
    *   *It will cover...*

### 3. Findings / Main Body (Hallazgos / Cuerpo Principal)
*   Divide esta sección en subapartados con títulos claros y descriptivos.
*   Presenta los hechos y datos de forma objetiva.
*   Analiza la información, sin expresar opiniones personales a menos que se solicite específicamente.
*   Utiliza listas, viñetas o numeración para mayor claridad.
    *   *According to the survey, ...*
    *   *It was observed that...*
    *   *The data suggests that...*

### 4. Conclusions (Conclusiones)
*   Resume los hallazgos más importantes de forma concisa.
*   Deriva las conclusiones lógicas a partir de la información presentada en la sección de "Findings". No introduzcas nueva información aquí.
    *   *In conclusion, it can be stated that...*
    *   *The evidence suggests that...*

### 5. Recommendations (Recomendaciones)
*   Basadas directamente en las conclusiones, ofrece sugerencias o acciones a tomar.
*   Deben ser prácticas y realistas.
*   Puedes usar frases como:
    *   *It is recommended that...*
    *   *I/We would suggest that...*
    *   *Action should be taken to...*

## Estructuras Gramaticales Avanzadas para Incluir

*   **Passive Voice:** Fundamental para un tono objetivo e impersonal (e.g., *The decision was made...*, *It is understood that...*).
*   **Reported Speech:** Para citar lo que otros han dicho o los resultados de encuestas (e.g., *Respondents indicated that...*).
*   **Modal Verbs:** Para hacer recomendaciones o sugerencias (e.g., *could be improved*, *should be considered*, *might be necessary*).
*   **Linking Words (Formal):** Para asegurar una progresión lógica y cohesión (e.g., *Furthermore, Moreover, Consequently, Therefore, In addition, However, Nevertheless*).
*   **Non-defining Relative Clauses:** Para añadir información contextual sin interrumpir el flujo principal.
