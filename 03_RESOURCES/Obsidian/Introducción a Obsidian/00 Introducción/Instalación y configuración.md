
> Pasos para tener Obsidian listo y bien configurado desde el principio.

## Instalación
1. Ve a [obsidian.md](https://obsidian.md) y descarga para Windows
2. Instala y abre
3. Clic en **"Create new vault"**
4. Nombre: `SecondBrain` (o el que quieras)
5. Elige la carpeta donde vivirá — ponla en un sitio que sincronices 
   (Google Drive, iCloud, OneDrive)

## Configuración inicial recomendada

### Apariencia

Settings → Appearance

- Theme: Minimal o Things (los más limpios)
- Font size: 16px mínimo

### Editor
Settings → Editor

- Readable line length: ON
- Spell check: ON (selecciona tu idioma)
- Auto pair brackets: ON
- Vim key bindings: OFF (a menos que uses Vim)

### Archivos y enlaces

Settings → Files & Links

- Default location for new notes: 00_Inbox
- Use [[Wikilinks]]: ON
- Detect all file extensions: ON


## Estructura de carpetas inicial
Crea estas 5 carpetas nada más crear el vault:
```

SecondBrain/ 
	├── 00_Inbox/ 
	├── 01_Projects/ 
	├── 02_Areas/ 
	├── 03_Resources/ 
	└── 04_Archive/

```
## Core Plugins que activar desde el inicio


Settings → Core Plugins


| Plugin | Para qué |
|--------|----------|
| Backlinks | Ver qué notas apuntan a la actual |
| Graph view | Visualizar conexiones |
| Templates | Plantillas de notas |
| Daily notes | Nota diaria automática |
| Quick switcher | Navegar entre notas con Ctrl+O |
| Tag pane | Panel de tags |

## Atajos esenciales

| Atajo | Acción |
|-------|--------|
| `Ctrl+N` | Nueva nota |
| `Ctrl+O` | Abrir nota (buscar) |
| `Ctrl+P` | Paleta de comandos |
| `Ctrl+E` | Alternar edición/preview |
| `Ctrl+G` | Abrir Graph View |
| `[[` | Crear link a otra nota |

## Primer paso tras instalar
No organices. Abre `00_Inbox` y escribe tu primera nota.
Cualquier cosa. El sistema se construye usándolo, no planificándolo.
