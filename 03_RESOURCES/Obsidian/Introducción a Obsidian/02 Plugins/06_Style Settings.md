## ¿Para qué sirve?
>Añade un panel visual con controles (sliders, toggles, colores) para personalizar el tema activo sin escribir CSS. Funciona con cualquier tema que lo soporte — **Minimal** es el más completo.

## Instalación
- Settings --> Community plugins --> Browse
- Buscar : Style Settings --> Install --> Enable

## Para elegir que tema usar
- Settings --> Appearance --> Manage
