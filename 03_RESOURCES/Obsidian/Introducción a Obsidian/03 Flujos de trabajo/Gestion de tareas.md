# Gestión de Tareas en Obsidian

Esta nota documenta cómo gestiono mis tareas utilizando el plugin `Obsidian Tasks` y el `Dashboard/Tareas`.

## Estructura de una Tarea

Las tareas siguen el formato:
`- [ ] Descripción #tag [priority:: high] [scheduled:: YYYY-MM-DD]`

## Visualización

Utilizo un dashboard central en `[[000_INBOX/Dashboards/Tareas]]` que agrega todas las tareas del vault.
