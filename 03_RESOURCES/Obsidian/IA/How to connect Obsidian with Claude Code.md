# How to connect Obsidian with Claude Code

## ¿Qué es?

Claude Code es una herramienta de Anthropic orientada a integrar Claude directamente en tu entorno de desarrollo y trabajo local. Su objetivo es permitir que la IA entienda la estructura de tus carpetas, archivos, proyectos y notas sin tener que copiar y pegar contexto constantemente en la versión web.

Combinado con Obsidian, puede convertirse en una especie de “segundo cerebro asistido”, ya que trabaja sobre tu sistema local y puede ayudarte a analizar, organizar, resumir, programar o mejorar contenido existente dentro de tu vault.

## ¿Cómo funciona?

El flujo habitual consiste en conectar Claude Code con un editor como Visual Studio Code, desde donde Claude puede interactuar con tus archivos locales.

En este caso:

- Obsidian actúa como base de conocimiento personal.
- Visual Studio Code funciona como entorno técnico para editar y automatizar.
- Claude Code sirve como asistente que entiende el contenido del sistema local.

Así puedes trabajar con notas Markdown, scripts, proyectos y documentación desde una misma estructura.

## ¿Cómo se usa?

### Configuración básica

1. Instalar Claude Code.
2. Instalar Visual Studio Code.
3. Añadir la extensión oficial de Claude Code en VS Code.
4. Abrir la carpeta de tu vault de Obsidian o proyecto local.
5. Dar permisos necesarios para lectura/escritura.

### Usos prácticos con Obsidian

- Resumir notas largas.
- Buscar conexiones entre ideas.
- Convertir apuntes desordenados en documentación limpia.
- Crear índices automáticos.
- Generar templates Markdown.
- Revisar textos.
- Crear scripts para automatizar organización del vault.
- Consultar tu conocimiento personal sin buscar manualmente.

### Ejemplo real

“Analiza mis notas sobre IA y crea un mapa de temas principales.”

“Busca todas las notas relacionadas con Obsidian y crea un resumen.”

## Notas personales

- Mucho más potente que usar solo chat web.
- Ideal si ya uso Obsidian diariamente.
- Puede ahorrar muchísimo tiempo.
- Interesante para proyectos grandes y TFG.
- Hay que organizar bien carpetas y nombres de notas.
- Potencial enorme si lo combino con tareas personales + estudio + proyectos.

## Ventajas

- Contexto persistente local.
- No repetir información constantemente.
- Acceso directo a archivos reales.
- Mejor flujo de trabajo.
- Automatización útil.

## Riesgos / cosas a vigilar

- Dar permisos solo a carpetas necesarias.
- Revisar cambios automáticos antes de aceptar.
- Mantener estructura limpia del vault.
- No depender totalmente de la IA.
