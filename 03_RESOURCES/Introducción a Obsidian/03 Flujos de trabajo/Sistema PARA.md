# Sistema PARA

> El método de organización más usado en segundos cerebros digitales.
> Creado por Tiago Forte. PARA = Projects, Areas, Resources, Archive.

## ¿Qué es PARA?
Un sistema de 4 carpetas que organiza **toda** la información según su utilidad práctica — no por tema.
La pregunta clave no es "¿de qué trata esto?"
sino **"¿para qué lo necesito?"**

## Las 4 carpetas

### 01_Projects — Proyectos

Cosas con **objetivo concreto** y **fecha límite**.
Tienen un final definido.

Ejemplos:

- TFG ThinkAndDoIt
- Rowing fantasy
- Preparar examen Cambridge

### 02_Areas — Áreas
Responsabilidades **continuas** sin fecha de fin.
Estándares que mantienes a lo largo del tiempo.

Ejemplos:

- Fitness / Entrenamiento
- Bajo
- Trabajo / Oracle OIC
- Idiomas

### 03_Resources — Recursos

Conocimiento de **referencia** que puede ser útil en el futuro. No tiene acción inmediata asociada.

Ejemplos:

- Obsidian (esta documentación)
- Oracle OIC
- Cine analisis 
- Bundesliga análisis

### 04_Archive — Archivo
Todo lo **inactivo**. Proyectos acabados, áreas abandonadas,
recursos que ya no usas.

Ejemplos:
- Proyectos entregados
- Notas de cursos terminados
- Ideas descartadas

## El flujo

Captura → 00_Inbox → Procesas → P / A / R / A

1. Todo entra por `00_Inbox` sin pensar
2. Una vez a la semana procesas el inbox
3. Para cada nota te preguntas:


¿Está asociada a un proyecto activo? → 01_Projects 
¿Es una responsabilidad continua? → 02_Areas 
¿Es conocimiento de referencia? → 03_Resources 
¿Ya no es relevante? → 04_Archive

## Diferencia clave Projects vs Areas

| Projects               | Areas            |
| ---------------------- | ---------------- |
| Tiene fin              | No tiene fin     |
| Aprobar TFG            | Estudios         |
| Correr 10K en mayo     | Fitness          |

Un proyecto **vive dentro de un área**.
Cuando el proyecto acaba → Archive.
El área sigue.

## Reglas de oro

- **No subcarpetas infinitas** — máximo 2 niveles de profundidad
- **La perfección es el enemigo** — mejor mover rápido que mover perfecto
- **Archive no es papelera** — puedes recuperar cualquier cosa
- **Inbox se vacía cada semana** — si no, pierde su función
