
# Componentes

1. Obsidian : Una aplicación compuesta por markdown para construir un segundo cerebro, es decir, una casa para tus notas.
2. Ollama : Es una simple, poderosa herramienta para ejecutar modelos LLM en tu maquina local. (IA).
3. Smart Connections Plugin: Es un plugin comunitario de obsidian que conecta tus notas con modelos de IA ejecutandose en Ollama permitiendo un RAG

>¿Que es un RAG? Es una técnica de inteligencia artificial que combina un modelo generativo (imaginate como CHAT GPT) con un sistema de búsqueda de información. Primero recupera datos relevantes de documentos, bases de datos o internet, y despúes genera una respuesta basada en esa información.

# Proceso de importación de la herramienta

### Paso1:

1. https://ollama.com/download descargar ollama para windows
2. Instalas el ejecutable que se te descarga
3. Comprobar su instalación abriendo una terminal y escribiendo Ollama list
4. Ensure your installation is up to date by running ollama -v
