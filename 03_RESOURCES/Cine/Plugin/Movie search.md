

> Plugin para crear notas de películas automáticamente
> con póster, reparto, director y ficha completa desde TMDB.
## Instalación

- Settings → Community plugins → Browse 
- Buscar: "Movie Search" 
- Install → Enable
- 
## Requisito: API Key de TMDB
El plugin necesita una API key gratuita de TMDB.

1. Ve a [developer.themoviedb.org](https://developer.themoviedb.org)
2. Crea una cuenta gratuita
3. Ve a Settings → API → Request an API Key
4. Copia tu API Key
