# Segundo cerebro

## ¿Qué es?
Un sistema personal para capturar, organizar y conectar todo lo que aprendo, veo y pienso.
Construido en Obsidian con el sistema PARA.

## ¿Qué sé hasta ahora?
- PARA organiza por utilidad práctica
- Los links entre notas valen más que las carpetas
- La revisión semanal es lo que mantiene el sistema vivo
- El Jardín Digital hace que las ideas maduren con el tiempo
- Dataview convierte el vault en una base de datos

## ¿Qué me falta por entender?
- Cómo integrar IA local cuando sea posible
- Zettelkasten — si me aporta algo sobre PARA
- Cómo migrar todas mis notas del móvil
